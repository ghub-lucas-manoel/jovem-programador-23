3. CSS:
Evita utilizar valores absolutos (ex. 500px) para dimensionar elementes, a não ser que faça sentido que ele tenha o mesmo tamanho independente das definições da janela/tamanho de tela do dispositivo. Dá preferência pra valores relativos (ex. 50%, 5rem, 1.8em);
Quanto maior vai ficando o arquivo, mais difícil fica pra dar manutenção nele. No caso do CSS, você pode utilizar mais de um arquivo por página, referenciando ele certinho no HTML. Tu pode utilizar dessa tática pra separar formatação de elementos chave de uma página em arquivos diferentes se o arquivo css principal da página estiver ficando muito grande.
Todas as págians estão usando o mesmo código CSS, mas cada página tem seu próprio arquivo. Nesse caso, você pode utilizar um único arquivo (ex. style.css), e referenciar esse mesmo arquivo no HTML de todas as páginas que utilizarem essa estilização.
