1. HTML
Recomendo que, para os arquivos das demais páginas fora o index, você utiliza o nome da página como nome do arquivo, para facilitar a identificação de cada arquivo nos momentos de manutenção do código.
No head do html da página de Contatos, você declarou o css da página antes do arquivo global, é importante deixar o arquivo global declarado antes do local.

2. CSS
O conteúdo dos arquivos style.css são os mesmos, inclusive com a mesma estilização sendo declarada no index.css. Nesse caso, você pode partir para passar essa estilização para o global.css, poupando tamanho do repositório e processamento de arquivos no carregamento da tela.
